# SLICER | URL Shortener
![alt text](https://github.com/kaanersoy/slicer-fun/blob/main/public/assets/slicer-logo-purp.svg)

It is a URL shortener to get shortened URLs.

## Used Tecnologies
- Heroku(Hosting)
- MongoDB(Database)
- NodeJS(Serving HTML files and API)
